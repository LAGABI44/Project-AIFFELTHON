## 💡프로젝트 소개
```
1️⃣ 주제 : 건강검진 데이터를 통해 질병 예측   
2️⃣ 데이터셋 : 국가건강보험 - 혈압·혈당 데이터   
3️⃣ 컬럼 : 수축기혈압, 이완기혈압, 공복혈당, BMI, 성별, 진료내역 
4️⃣ 모델 : LightGBM + Random Forest  
5️⃣ 간단 설명 : 건강검진 데이터를 통해 정상과 비정상(질병)을 분류한다. 비정상은 어떤 질병을 가졌는지 2차 분류 후 관련 영양제를 추천해준다.
```
