## 💡프로젝트 소개
```
1️⃣ 주제 : 신체 검사 결과로 영양 상태 분석 후 영양 성분 추천   
2️⃣ 데이터셋 : 건강보험공단 - 건강검진결과 데이터   
3️⃣ 전처리 : AST, ALT, 혈색소, 공복혈당, 키, 몸무게만 사용   
4️⃣ 모델 : SVM, 추천 시스템   
5️⃣ 간단 설명 : 신체 검사 결과 자료에서 필요한 데이터만 추출한 후 SVM을 통해 1차 분류를 한다. 분류된 데이터로 추천시스템 로직을 통해 필요한 영양제를 추천한다.
```
